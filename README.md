# AnyFeast Onboarding Redesign

This project is a Flutter-based onboarding flow redesign created as part of the AnyFeast internship assignment.

## Focus areas
1) Reduced onboarding friction
2) Grouped related user inputs
3) Showed value early via meal plan preview
4) Added meaningful skip options

## Tech stack
1) Flutter
2) Dart

## Notes
This is a UX-focused redesign. The UI is intentionally kept simple to highlight flow and usability decisions.
